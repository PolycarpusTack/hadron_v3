Present tense (not conditional)British English"Check box" = two words
  1. Title — Always start with "Fix for". Concise and descriptive.
  2. Issue Description — Start with "Previously, ..." Describe what was wrong.
  3. Steps to Reproduce — Optional. Only for complex crashes.
  4. Cause — Why the issue occurred. Technical details if needed.
  5. Solution — What was done to fix it (if applicable).
  6. References — Ticket number in brackets: "(MGXPRODUCT-15754)"
Example:
Fix for episode title formulas for episodes with many transmissions
Previously, a crash would occur when defining the episode title formula with a placeholder on the episode of a series and executing the calculation of the placeholders.
This happened because the tag retrieved all transmissions and picked the first. If there were many, the query would go out of memory.
This issue has been fixed by adjusting the way titles are calculated to prevent memory overload. (MGXPRODUCT-15754)
